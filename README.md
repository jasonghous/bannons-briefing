# Bannons Acquisition - Strategic Briefing

Strategic briefing presentation on the acquisition of Desmond L Bannon & Sons funeral services in Liverpool.

## Deployment

This is a static HTML presentation ready for deployment to Vercel.

### Deploy to Vercel

1. Push this folder to a GitHub repository
2. Connect the repository to Vercel
3. Deploy (no build configuration needed - it's a static HTML file)

### Local Preview

Simply open `index.html` in your browser.

## Content

3-slide presentation covering:
- **Slide 1:** Strategic overview and acquisition rationale
- **Slide 2:** How Bannons fits the wider proposition (5 package offer, strategic sequence)
- **Slide 3:** Financial analysis (volumes, balance sheet, security filings)
